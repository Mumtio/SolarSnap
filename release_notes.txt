SolarSnap Android App - Release Notes
Version: 1.0.0
Build Date: March 13, 2026
Package: com.solarsnap.app
Target SDK: 36 (Android 14)

=== RELEASE OVERVIEW ===
Production release of SolarSnap Android application for FLIR ACE thermal camera integration. This is a professional solar panel inspection system designed for field technicians conducting thermal inspections of solar farms.

=== NEW FEATURES ===
• Complete thermal inspection workflow with FLIR ACE SDK 2.17.0 integration
• Real-time thermal imaging with live temperature measurement and hotspot detection
• Dual-view display (thermal + visual) for comprehensive panel analysis
• GPS-based location tracking for accurate fault positioning
• Interactive site mapping with up to 1,200+ panel visualization
• Offline-first operation with automatic cloud synchronization
• Comprehensive inspection history with search and filtering capabilities
• Barcode scanning for panel ID verification
• Enterprise-grade security with JWT authentication
• Role-based access control (Inspector, Manager, Admin)
• Background sync service for seamless data upload
• ACE-compliant UI optimized for 4" field devices with high contrast design
• Physical button navigation support for glove-friendly operation

=== TECHNICAL SPECIFICATIONS ===
• Minimum Android Version: API 33 (Android 13)
• Target Android Version: API 36 (Android 14)
• Architecture: ARM64-v8a (64-bit)
• Java Version: 21 (JDK 21 required for compilation)
• FLIR SDK: Atlas Android SDK 2.17.0
• Build Type: Release (signed, optimized, obfuscated)
• APK Size: ~95MB (includes FLIR SDK libraries)
